# Code standaarden

## Bestandsnamen
Gebruik overal:
- index.html
- app.js
- config.json
- README.md

## Taal
Bestands- en mapnamen in het Nederlands waar mogelijk.

## CSS
Gebruik zoveel mogelijk gedeelde CSS uit assets/styles.

## JavaScript
Zet toolspecifieke logica in de eigen app.js.
Zet gedeelde helpers in assets/scripts.

## JSON
Gebruik config.json voor:
- titels
- beschrijvingen
- dropdownopties
- feature toggles
