# Introductie

## Doel van de EMS Tool

De EMS Tool is een modulaire verzameling webtools voor het ondersteunen van EMS-gerelateerde workflows binnen een RP-omgeving. De focus ligt op het ondersteunen van behandelingen, rapportering, aanvragen, medische hulpmiddelen en administratieve processen.

De tools zijn gebouwd om:

- snel bruikbaar te zijn
- eenvoudig uitbreidbaar te blijven
- zonder backend te werken
- afzonderlijk te kunnen functioneren
- makkelijk te documenteren en onderhouden te zijn

---

## Voor wie deze tool bedoeld is

De tool is bedoeld voor:

- EMS-leiding en beheer
- medewerkers die rapporten of aanvragen opmaken
- ontwikkelaars of beheerders die modules aanpassen
- mensen die de logica en opbouw van de tool willen begrijpen

---

## Kernprincipes

### Modulair
Elke tool staat op zichzelf. Er is geen verplichte koppeling tussen modules.

### Statisch
De tool werkt zonder backend, database of centrale opslag.

### Begrijpbaar
De code moet leesbaar zijn en voldoende commentaar bevatten.

### Praktisch
De tools zijn bedoeld als hulpmiddelen voor concrete acties: invoer, verwerking, output.

### Consistent
De algemene structuur, layout en documentatiestijl worden zoveel mogelijk gelijk gehouden.

## Wat deze wiki niet is

Deze wiki is geen technische API-documentatie voor een backend, omdat het project geen backend gebruikt. Het is ook geen gebruikershandleiding voor één enkele tool, maar een centrale kennisbank voor de volledige toolset.
