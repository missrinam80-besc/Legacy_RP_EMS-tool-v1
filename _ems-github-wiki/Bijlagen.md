# Bijlagen

## Woordenlijst

### Module
Een zelfstandige tool met eigen bestanden en eigen doel.

### Centrale styles
CSS-bestanden die door meerdere modules gedeeld worden.

### Configuratie
Instelbare data zoals labels, opties of mappings, meestal in JSON.

### Output
Het resultaat van een module, zoals een rapporttekst, samenvatting of berekening.

---

## Voorbeeld basisstructuur module

```text
/tools/module-naam/
  index.html
  app.js
  config.json
  README.md
```
