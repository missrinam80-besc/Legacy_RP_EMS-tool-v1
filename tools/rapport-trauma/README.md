# Trauma Rapport

Basisrapporttool voor trauma-interventies.

## Bestanden
- index.html → interface
- app.js → logica
- config.json → instellingen en keuzelijsten

## Werking
De gebruiker vult de velden in, waarna een rapporttekst wordt opgebouwd.
Deze kan gekopieerd of als .txt gedownload worden.

## Uitbreidbaar met
- vitale functies
- pijnscore
- bloedverlies
- per ledemaat letsels
- nazorginstructies
