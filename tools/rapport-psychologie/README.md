# Psychologisch Verslag

Rapportmodule voor intake, crisisopvang, psychologische consulten en opvolging.

## Bestanden
- index.html → interface
- app.js → logica en rapportopbouw
- config.json → keuzelijsten en instellingen
- psychologie-rapport.css → modulespecifieke styling

## Onderdelen
- patiëntgegevens
- consultgegevens
- behandelaar en betrokkenen
- mentale inschatting
- traject en advies
- opvolging en kosten
- tekstlog en extra notities
