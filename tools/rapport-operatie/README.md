# Operatie Rapport

Basisrapporttool voor operaties en chirurgische ingrepen.

## Bestanden
- index.html → interface
- app.js → logica
- config.json → instellingen en keuzelijsten
- operatie.css → modulespecifieke styling

## Werking
De gebruiker vult de velden in, waarna een gestructureerd operatierapport wordt opgebouwd.
Deze kan gekopieerd of verder gebruikt worden in de EMS tablet.

## Onderdelen
- patiëntgegevens
- operatiegegevens
- operatieteam
- pre-operatieve toestand
- anesthesie en voorbereiding
- uitgevoerde ingreep
- post-operatief plan
- kosteninformatie
- medisch log
- extra notities
