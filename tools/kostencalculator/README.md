# Kostencalculator

Tool om één of meerdere medische onderdelen te selecteren en automatisch een totaalkost te berekenen.

## Bestanden
- index.html → interface
- app.js → logica
- config.json → lijst van behandelingen en prijzen

## Werking
De gebruiker vinkt behandelingen aan, waarna de tool:
- het totaal berekent
- een korte samenvatting opbouwt
- kopiëren en downloaden mogelijk maakt
