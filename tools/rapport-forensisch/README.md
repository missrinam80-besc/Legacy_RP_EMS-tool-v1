# Forensisch Rapport

Basisrapporttool voor medische en forensische vaststellingen.

## Bestanden
- index.html → interface
- app.js → logica
- config.json → instellingen en keuzelijsten
- forensisch.css → modulespecifieke styling

## Werking
De gebruiker vult de velden in, waarna een gestructureerd forensisch rapport wordt opgebouwd.
Deze kan gekopieerd of verder gebruikt worden in de EMS tablet.

## Onderdelen
- patiënt / betrokkene
- algemene forensische gegevens
- onderzoeksteam
- toestand van betrokkene
- letsels en uiterlijke vaststellingen
- forensische bevindingen
- stalen en bewijsstukken
- besluit en overdracht
- kosteninformatie
- medisch log
- extra notities
