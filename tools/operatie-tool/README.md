# Operatie-assistent

Referentiemodule voor chirurgie.

## Doel
Deze tool bundelt de chirurgische workflow in één view:
- casus en indicatie
- pre-op screening
- operatie-checklist
- benodigdheden en RP-stappen
- post-op plan
- chirurgisch kostenblok

## Gebruik
1. Vul de casus in
2. Vul pre-op screening aan
3. Vink checklistpunten af
4. Noteer ingreep en nazorg
5. Genereer samenvatting
6. Gebruik output als basis voor het operatierapport
