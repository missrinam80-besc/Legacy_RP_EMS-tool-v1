# Beheer modules

Centrale beheermodule voor zichtbaarheid, status en basisinstellingen van de EMS-modules.

## Storage
- key: `ems_admin_modules`
- defaults: `data/admin/default-modules.json`

## Functies
- modules oplijsten
- filters op type, afdeling, status en zichtbaarheid
- toevoegen, bewerken en verwijderen
- import/export JSON
- reset naar standaard

## Doel
Deze module maakt het mogelijk om centraal te bepalen welke tools, portalen, rapporten en beheerschermen zichtbaar en actief zijn.
