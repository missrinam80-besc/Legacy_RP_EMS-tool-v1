# Beheer thema

Centrale beheermodule voor branding, kleuren en lay-outinstellingen van de EMS-suite.

## Storage
- key: `ems_admin_theme`
- defaults: `data/admin/default-theme.json`

## Functies
- algemene sitegegevens bewerken
- kleuren en visuele instellingen aanpassen
- live preview tonen
- import/export JSON
- reset naar standaard

## Opmerking
Het thema wordt in v1 via `theme-loader.js` toegepast op pagina's die deze loader inladen.
