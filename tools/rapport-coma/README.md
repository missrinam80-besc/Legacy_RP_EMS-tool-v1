# Coma Rapport

Basisrapporttool voor coma-opname, neurologische observatie en opvolging.

## Bestanden
- index.html → interface
- app.js → logica
- config.json → instellingen en keuzelijsten
- coma.css → modulespecifieke styling

## Werking
De gebruiker vult de velden in, waarna een gestructureerd coma rapport wordt opgebouwd.
Deze kan gekopieerd of verder gebruikt worden in de EMS tablet.

## Onderdelen
- patiëntgegevens
- algemene opnamegegevens
- arts en team
- neurologische toestand
- vitale functies
- oorzaak en letsels
- onderzoeken en behandeling
- observatieverloop
- vervolgbeleid
- kosteninformatie
- medisch log
- extra notities
