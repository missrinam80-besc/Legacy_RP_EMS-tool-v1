# Opname Rapport

Basisrapporttool voor opnames en hospitalisaties.

## Bestanden
- index.html → interface
- app.js → logica
- config.json → instellingen en keuzelijsten

## Werking
De gebruiker vult de velden in, waarna een gestructureerd opnamerapport wordt opgebouwd.
Deze kan gekopieerd of verder gebruikt worden in de EMS tablet.

## Onderdelen
- patiëntgegevens
- opnamegegevens
- arts en assistenten
- klinische toestand
- voorlopige diagnose
- observaties
- opgestarte zorgen en medicatie
- opnameplan
- kosteninformatie
- medisch log
- extra notities
